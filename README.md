## Project Overview
**Campus Room Finder** aims to simplify how students, staff, and administrators locate and manage university rooms.  
Users can search for available classrooms, labs, or study spaces in real time, while admins can manage room data via a secure backend dashboard.

## Key Features
- **Room Search:** Search by name, building, capacity, or features.  
- **Interactive Map:** View rooms on a campus map for easier navigation.  
- **Availability Tracking:** See which rooms are currently in use or free.  
- **Admin Panel:** Add, update, or remove room information.  
- **Responsive Design:** Works smoothly on desktop and mobile devices.
